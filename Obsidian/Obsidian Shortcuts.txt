Obsidian noting app shortcuts:
Switch to Reading/Editting mode		CTRL + E
close tab				CTRL + W
create new note				CTRL + N
go to a file or create new one		CTRL + O
make text bold         CTRL + B
line break            SHIFT + ENTER

heading        up to 6 #

italic      *text*    _text_
strikethrough    ~~text~~

escape character    \

external image     ![text](link)

quote          > text
callout       [!info] ain quote
