java به دلیل استفاده از ماشین مجازی کند تر می شود. می توان برای افزایش سرعت:
- استفاده از کد ماشین خاصی
- استفاده از کامپایلر های just-in-time

جاوا قابلیت اجرا در word wide web دارد (این برنامه ها applet نام دارند)

- ساخت یافته structured approach
نام های دیگر: top-down design, bottom-up design, stepwise refinement, modular design
- شیء گرا object oriented apptraoch

تبدیل انواع:
- در عبارات محاسباتی
	- انواع کوچک تر به انواع بزرگ تر تبدیل می شوند
	double > float > long > int > short
- در دستورات انتساب (type casting)
	y = (falot) x

رشته فاقد هرگونه کاراکتر: empty یا null ("")

string concatenation (+): اتصال استرینگ ها به هم یا به اعداد

متغیر ها:
متغیر های نوع داده اولیه: از نوع داده های اولیه primary
متغیر های مرجع (refrence variable): تعریف شده از یک کلاس

هر کلاس باید در داخل پکیجی (package) قرار گیرد. هر پکیج می تواند چندین کلاس داشته باشد. اگر کلاسی در یک پکیج قرار نگیرد جاوا خودش یکی می سازه
کلاس ---> پکیچ ---> کتابخانه

انتزاع: استخراج داده های ضروری مسئله و ارتباط بین آن ها از صورت مسئله
